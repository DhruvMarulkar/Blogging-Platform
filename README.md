# 🚀 FastAPI Blogging Platform

A production-ready blogging platform built with **FastAPI**, featuring JWT authentication, password reset via email, image uploads, Docker support, PostgreSQL, Alembic migrations, and Amazon S3 integration.

---
## 🌐 Live Demo

>**http://13.235.17.185:8080/**

---
## ✨ Features

- 🔐 JWT Authentication
- 👤 User Registration & Login
- 📝 Create, Edit & Delete Blog Posts
- 🖼 Upload Profile Pictures
- ☁ Amazon S3 Media Storage
- 🔄 Password Reset via Email
- 🔒 Secure Password Hashing
- 📄 Pagination
- 🐳 Dockerized Deployment
- 🗄 PostgreSQL Database
- 🔁 Alembic Database Migrations
- ⚡ Async SQLAlchemy
- ✅ Unit & API Tests with Pytest
